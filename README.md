# BA Wireframe UI

**Wireframe tĩnh (low-fi) miễn phí – Dành riêng cho Business Analyst**

Bộ sưu tập component wireframe **xám xịt, hoàn toàn tĩnh**, giúp BA prototype nhanh, annotate dễ dàng và thảo luận rõ ràng với team, dev, PO & khách hàng.

### Tại sao nên dùng?
- ✅ Copy-paste 100% (HTML + Tailwind CSS thuần)
- ✅ Không cần cài đặt, không animation, không JS
- ✅ Thiết kế low-fi chuẩn BA (dễ highlight, comment, screenshot)
- ✅ Responsive cơ bản (desktop + mobile)
- ✅ Dùng ngay cho: User Flow, Requirement Document, Stakeholder Review, Wireframe Workshop

### Các component hiện có
- Navbar
- Hero Section
- Features Grid
- Pricing Table
- Login / Registration Form
- Dashboard Layout
- Skeleton & Placeholder
- Footer + nhiều component khác (sẽ update liên tục)

### Cách sử dụng
1. Vào thư mục `components/`
2. Copy file HTML bạn cần
3. Dán vào dự án của bạn (hoặc mở trực tiếp)
4. Thêm ghi chú BA ngay trong code (ví dụ: `<!-- BA Note: Field này bắt buộc -->`)
